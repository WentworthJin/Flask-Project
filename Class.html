Class and Object ( Instances )

Just like many other object-orinted programming languages, the Class is the building blocks of your program's code in Swift. In Swift, you can define properties and methods to add functionality to your classes, with the same syntax you use to define constants, variables, and functions. You can learn how the Class and object works in the following examples.


The relationship between Class and Object ( Instances )

In general, the Object ( also called Instances in Swift ) are created from Class, and the Object represents individual instances of the Class. As for the Class, it represent a category of things, which includes the propertites and features it need.

Overall, the Class classified the range of features and propertites, and the Object is created has different features within the given range. You can learn more details in the following sections.

What is Class

Similar to other object-orinted programming languages, the Class describes the kind of properties and categories it has, for example, we have a class Student { }, and all the features and properties of a Student are stored in the class, such as Name, mark, and more.

In Swift, the Class has three major components: Class name, declare the variable name, and declare the variable type. For example, as you can see the picture below, the Class has a name "student", three declared variable ( var) with name, and their vaule types ( Int & String). The value types decide the kind of Data types it can be, for example, the studname variable can only be string types of data. 

Class student {
  var studentname: String
  var mark: Int
}

What is a Object ( Instances )

As mentioned previous, the Instace is an individual case that created from the Class.

Taking the previous Class "student" as an example, you can create an Instance of that Class by using: 

Class student {
  var studentname: String = "mark"
  var mark: Int = 0
}

This Instance represent one student, who's name is Mark, and his mark is 90

In addition, you can assign a variable to this Instance, in order to access it's properties.

For example, let marks = student( ). Then you can use this variable to access all the properties & functions inside that Instance.

In addition, you can also use this variable to change some properties of that Instance. As the picture shows below, you can change the "mark" value to 100. Then if you use the print function, you can see the mark has been changed to 100 now. 


let marks = student()
marks.mark = 100
marks.studname = "Mark"

print(marks.amrk)
