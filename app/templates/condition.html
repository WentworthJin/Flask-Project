Conditionals & Control Flow

Swift provides a variety of conditions & control flow statements. These include while loops to perform a task multiple times, in addition, the if, guard, and switch statements are used to execute different branches of code based on certain conditions. What's more, like many other programming languages, it has break and continue statements that can transfer the flow of execution to another point in your code.

Conditionals

Swift provides two ways to add conditional branches to your code, which is the if statement and the switch statement. In general, the if statement is used to evaluate simple conditions with only a few possible outcomes, while the switch is better suited to more complex conditions that comes with multiple possible premutations.

If-Else

Similar to other programming languages, the If-Else conditional statements can be divided into two part: If and Else

In the simplest form, the if statement has a single if condition, and it executes a set of statements if the specified condition is True. The basic grammar of if statement in Swift is described below. 

      if < something is True > {
         /* Execute the code */
      }
    
    
      Example
    
      var Temp = 30
      if Temp <= 32 {
        print("It's very cold. Consider wearing a scarf.")
      }
      
      // Prints "It's very cold. Consider wearing a scarf."

The if statement provides an alternative set of statements: else clause, for situations when the if condition is false. These statements are indicated by the else keyword. 

      if < something is True > {
         /* Execute the code */
      } else {
         /* Execute the alternative code if condition is false */
      }
    
    
      Example
    
      var Temp = 20
      if Temp >= 28 {
        print("It's very hot.")
      } else {
        print("It's very cold. Consider wearing more clothes.")
      }
      
      // Prints "It's very cold. Consider wearing more clothes."

Switch

The Switch statement in Swift is similar to the If-Else statement. However, it condsiders a value and compares it with several possible matching patterns. In addition, if the value cannot match any patterns, it can provide a default result. The simplest grammar of switch is presented below. 

      switch some value to consider {
        case value 1:
            respond to value 1
        case value 2:
            respond to value 2
        default:
            otherwise, do something else
        }

While Loops

In Swift, a while loop performs a set of statement until a condition becomes a false. Normally, the while loop is used when the number of iterations isn't known before the first iteration begins. 

      while condition {
        statements
      }
