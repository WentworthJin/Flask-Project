Swift environment setup

Swift is an open source programming language used to develop OS X and iOS applications.

Before formally developing an application, we need to build a Swift development environment so that we can use various development tools and languages for rapid application development in a better and friendly manner. Since the Swift development environment needs to run in the OS X system, the construction of its environment will be different from the Windows environment. Let's learn how to build the swift development environment.

Prerequisites for successfully setting up a swift development environment:

What you need

  • Must have an Apple computer. Because the integrated development environment XCode can only run on OSX system.
  • The computer system must be OS 10.9.3 and above.
  • The computer must install the Xcode integrated development environment.

Swift development tool Xcode download

Swift development tools? Official website address: https://developer.apple.com/xcode/download/

The first Swift program (Try it now!!!)

After Xcode is installed, we can start writing Swift code.

Next, we open Xcode in the application folder, and select File => New => Playground at the top of the screen after opening.


Then set a name for the playground and select the iOS platform.

UIKit is a kind of basic package, applied in interactive IO streams with mobile devices.

                import UIKit
                var str = "Hello, playground"

If you want to create an OS x program, you need to import the Cocoa package. The import Cocoa code is asfollows: 

                import Cocoa
                var str = "Hello, playground"

Run it, then the output shows.

Create the first project

1. Open the xcode tool, select File => New => Project

2. We select a "Single View Application" and click "next" to create a simple sample app application.

3. Then we enter the project name (ProductName), company name (Organization Name), company identifier prefix name (Organization identifier), select the development language (Language). Among them, Language has two options: Objective-c and swift, because we are learning swift, of course, we choose swift. Click "Next" to the next step.

4. Select the storage directory. If you want to use Git source code management, tick create git repository on My Mac under Source Control. Click create to create the project.

5. After the project is created, a sample file is generated by default. You can see that swift merges the h and m files in oc into one file (that is, the swift suffix file). Main.storyboard is equivalent to the xib file, which is more than xib Many functions.

6. Open main.storyboard and you will see a simple blank application interface by default, the size of which is the size of a tablet interface. If you only need to develop an iPhone compatible app, you can uncheck the Use Auto Layout check box (the default is to check it).

7. A dialog box pops up, let us choose the interface size, iPhone or iPad. We choose the size of the iPhone.

8. We add some content to the interface, find the Text control at the bottom right, drag it onto the storyboard, and double-click to write the text "Hello World!".

Run the simulator (command+R shortcut or select Product => Run in the menu bar).

Congratulation!!!
